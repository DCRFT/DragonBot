# DragonBot
A Discord bot for DragonCraft Minecraft server.

Features:\
• link Minecraft account to Discord server,\
• moderation commands,\
• bot status,\
• YouTube music player,\
• tickets/suggestions system with votes.\
• role menu.\
• auto user voice channels,\
• and more...
